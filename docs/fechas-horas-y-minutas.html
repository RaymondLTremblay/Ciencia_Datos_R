17.4 Intervalos y duraciones

La función interval() crea un objeto R de intervalo de tiempo. En este caso, el intervalo entro la fecha en la tabla, y el día de hoy. Nota que esto puede ser muy util si quiere calcular la cantidad de tiempo entre dos fechas.

Code 
nueva_fechas %>%
  mutate(intervalo = interval(fecha, today()))
## # A tibble: 4 × 3
##   codigo fecha               intervalo                              
##   <chr>  <dttm>              <Interval>                             
## 1 001    2019-01-01 00:00:00 2019-01-01 00:00:00 UTC--2023-08-21 UTC
## 2 002    2019-03-31 01:05:00 2019-03-31 01:05:00 UTC--2023-08-21 UTC
## 3 003    2019-06-14 20:00:00 2019-06-14 20:00:00 UTC--2023-08-21 UTC
## 4 004    1859-03-01 11:32:13 1859-03-01 11:32:13 UTC--2023-08-21 UTC

int_length() regresa el número de segundos dentro del intervalo

Code 
nueva_fechas %>%
  mutate(
    intervalo = interval(fecha, today()),
    segundos = int_length(intervalo)
    )
## # A tibble: 4 × 4
##   codigo fecha               intervalo                                 segundos
##   <chr>  <dttm>              <Interval>                                   <dbl>
## 1 001    2019-01-01 00:00:00 2019-01-01 00:00:00 UTC--2023-08-21 UTC  146275200
## 2 002    2019-03-31 01:05:00 2019-03-31 01:05:00 UTC--2023-08-21 UTC  138581700
## 3 003    2019-06-14 20:00:00 2019-06-14 20:00:00 UTC--2023-08-21 UTC  132033600
## 4 004    1859-03-01 11:32:13 1859-03-01 11:32:13 UTC--2023-08-21 UTC 5190265667

nueva_fechas %>%

17.6 as.duration

as.duration() crea un objecto en R que contiene la duración del intervalo de tiempo.

Code 
nueva_fechas %>%
  mutate(desde_hoy = as.duration(fecha %--% today()))
## # A tibble: 4 × 3
##   codigo fecha               desde_hoy                  
##   <chr>  <dttm>              <Duration>                 
## 1 001    2019-01-01 00:00:00 146275200s (~4.64 years)   
## 2 002    2019-03-31 01:05:00 138581700s (~4.39 years)   
## 3 003    2019-06-14 20:00:00 132033600s (~4.18 years)   
## 4 004    1859-03-01 11:32:13 5190265667s (~164.47 years)

El objeto de duración de tiempo se puede filtrar fácilmente basado en una variedad de tipo de tiempos. En este caso, semanas. dweeks() crea un objeto de duración de la largura especificada

Code 
nueva_fechas %>%
  mutate(desde_hoy = as.duration(fecha %--% today())) %>%
  filter(desde_hoy > dyears(100))
## # A tibble: 1 × 3
##   codigo fecha               desde_hoy                  
##   <chr>  <dttm>              <Duration>                 
## 1 004    1859-03-01 11:32:13 5190265667s (~164.47 years)

Otra opción es ddays().

Code 
nueva_fechas %>%
  mutate(desde_hoy = as.duration(fecha %--% today())) %>%
  filter(desde_hoy < ddays(700))  # Note that you filter for the number of days that have acculmulated
## # A tibble: 0 × 3
## # ℹ 3 variables: codigo <chr>, fecha <dttm>, desde_hoy <Duration>

hacer_fechahora_100 <- function(anio, mes, dia, tiempo) {
  make_datetime(anio, mes, dia, tiempo %/% 100, tiempo %% 100)
